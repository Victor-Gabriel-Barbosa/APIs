Sobre a API Ninja

A API Ninja oferece acesso a diversas APIs úteis e divertidas. Para usar este testador:

  1. Obtenha sua chave gratuita em api-ninjas.com
  2. Cole sua chave no campo "Chave da API"
  3. Ative a "Tradução Automática" se desejar conteúdo em português
  4. Selecione uma das APIs disponíveis
  5. Clique em "Testar API" para ver o resultado

Dica: Suas configurações são salvas localmente no navegador para sua conveniência.

Tradução: Os resultados são automaticamente traduzidos do inglês para português quando ativado.
